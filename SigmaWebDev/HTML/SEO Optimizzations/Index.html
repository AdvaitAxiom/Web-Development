  • Core Web Vitals
    CLS(Cumulative Layout Shift)

    CLS is a measure of the largest burst of layout shift scores for every unexpected layout shift that occurs duing the entire lifespan of a page.

    We should add height and width to our images to make browser understand how much space should be left for the image otherwise it will shift it unethically

    LCP(Larger contentful Paint)

    To provide a good user experience, sites should strive a have LCP of 2.5 sec or less.(With in that time the lagest part of the webpage/ content should be loaded)

    FID(First Input Delay)

    Delay that be counted after user interacts with any input elements in the website and it should be around 100 milliseconds or less
  • Lighthouse
